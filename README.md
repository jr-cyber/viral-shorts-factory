# 🎬 Viral Shorts Factory v1

Create and manage viral short-form content (TikTok, Reels, Shorts, etc.)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## TikTok Integration

1. Get credentials at https://developers.tiktok.com/
2. Copy `.env.example` to `.env.local` and add your keys
3. Connect via the app UI

## Features

- Video creation interface
- TikTok OAuth + publishing API
- Analytics dashboard
- Template library (coming soon)
